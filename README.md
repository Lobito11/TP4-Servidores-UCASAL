**TRABAJO PRÁCTICO ACTIVIDAD OBLIGATORIA  4:**

Modalidad: Grupal

Despliegue de un servidor en la nube

_Debe ser enviada para su evaluación_

La presente práctica tiene como objetivo configurar un **workspace** en **Github Codespace** que permita desplegar un servidor de pruebas Docker in Docker. 

Conceptos generales

Responder a las siguientes preguntas:

1)        Según la clasificación de los modelos de computación en la nube, (IaaS, PaaS, SaaS). ¿A cuál de los tres pertenece **Github Codespaces**? ¿Por qué?

2)        Describa y grafique la arquitectura y componentes principales de **Github Codespaces**.

3)        ¿Qué tipo de entornos permite desplegar **Github Codespaces**? ¿Qué limitaciones tiene para su uso de manera gratuita?

4)        ¿Cuál es la configuración mínima necesaria para la configuración de un **workspace**?

5)        Cuál es la función del archivo de configuración **devcontainer.json**? ¿Dónde se crea?

6)        Genere un archivo de configuración **devcontainer.json** que permita el despliegue de un **workspace** **Docker in Docker**. 

7)        Con el archivo generado en el punto anterior, desplegar el **workspace** en la nube. 

Modo de presentación:

·        Deberá subir a la plataforma un archivo **PDF** con las respuestas a los puntos 1 a 6.

·        Grabar y subir un enlace a un **video** que muestre el despliegue del **workspace** en la nube.
